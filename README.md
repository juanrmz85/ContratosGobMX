Análisis de Datos de Contratos | Bootcamp BEDU (2022)

Descripción: 
Realicé una limpieza y análisis exploratorio de datos a una gran base de datos de contratos del Gobierno Federal, esto con el objetivo de identificar aquellas oportunidades de las dependencias del gobierno federal para hacer un manejo adecuado de los recursos.

Con este ejercicio, fue posible practicar actividades de limpieza y análisis a una gran base de datos de una fuente real.

Herramientas: 
Python | Pandas | Numpy
